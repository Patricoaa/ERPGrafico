# 0001 — Next.js App Router

## Context
We need a robust framework for building the frontend of ERPGrafico with server-side rendering, routing, and modern React features.

## Decision
Use Next.js 16 with the App Router architecture.

## Consequences
- **Positive:** Improved performance, built-in routing, server components.
- **Negative:** Steeper learning curve for developers used to the Pages Router.

## Alternatives considered
- React SPA (Vite)
- Next.js Pages Router
